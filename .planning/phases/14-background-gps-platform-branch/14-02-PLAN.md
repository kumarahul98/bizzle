## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| GPS stream → main isolate | On iOS the position stream now runs in the SAME process as the UI (no isolate boundary). Raw `Position` lat/lng (PII) is in-process. |
| geolocator platform error → UI | Mid-trip stream errors may carry platform diagnostics including coordinates. |
| temporary-full-accuracy purpose | The purpose string is shown to the user by iOS; it must not leak internal data. |

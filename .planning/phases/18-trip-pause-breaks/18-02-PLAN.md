<objective>
Teach the trip pipeline to pause. `TripAccumulator` gains a pause/resume model: while paused it keeps drawing the polyline but stops attributing distance and moving/stuck time, and it freezes the displayed timer. `finalize` closes any open break, computes ACTIVE duration (wall-clock âˆ’ total paused), and emits the break segments on the `FinalizedTrip` DTO. The persist path writes those segments into the `trip_breaks` table (Plan 18-01) plus the `total_paused_seconds` aggregate on the trip â€” all inside the existing atomic transaction (TRACK-09).

Purpose: this is the pure-logic heart of the phase. Paused time MUST be excluded from duration and the moving/stuck breakdown (SC#2), and the timer MUST freeze the instant pause fires (D-06) so the UI in Plan 18-03 is a dumb terminal with no local pause clock. Breaks persist as normalized rows (Plan 18-01's table) so Phase 19 can edit them. No cross-isolate wiring or UI here â€” that is Plan 18-03. ONLY primitive maps cross the isolate boundary, so the new snapshot/DTO fields are added as primitives in `toMap`/`fromMap`.
Output: a pause-aware accumulator, an extended `FinalizedTrip` DTO, a persist path that writes breaks + the aggregate, and a heavy TDD suite proving paused time is excluded everywhere.
</objective>

```dart
class TripAccumulator {
  TripAccumulator({required this.startedAt}) : _tripId = const Uuid().v4();
  final DateTime startedAt;
  final String _tripId;
  Position? _lastAccepted; DateTime? _lastAcceptedAt;
  double _distanceMeters = 0; int _timeMovingSeconds = 0; int _timeStuckSeconds = 0;
  final List<Position> _samples = <Position>[]; bool _finalized = false;

  void addSample(Position p) {
    if (_finalized) return;
    if (p.accuracy > kTrackingMaxAcceptableAccuracyMeters) return;
    final prev = _lastAccepted;
    if (prev == null) { _lastAccepted = p; _lastAcceptedAt = p.timestamp; _samples.add(p); return; }
    final deltaMillis = p.timestamp.difference(prev.timestamp).inMilliseconds;
    if (deltaMillis <= 0) { _samples.add(p); _lastAccepted = p; _lastAcceptedAt = p.timestamp; return; }
    final deltaSec = deltaMillis / 1000.0;
    _distanceMeters += Geolocator.distanceBetween(prev.latitude, prev.longitude, p.latitude, p.longitude);
    if (deltaSec <= kTrackingMaxAttributableGapSeconds) {
      final deltaSecInt = deltaSec.round();
      if (prev.speed >= kStuckSpeedThresholdMs) { _timeMovingSeconds += deltaSecInt; }
      else { _timeStuckSeconds += deltaSecInt; }
    }
    _lastAccepted = p; _lastAcceptedAt = p.timestamp; _samples.add(p);
  }

  TripSnapshot snapshot(DateTime now) { ... elapsedSeconds: now.difference(startedAt).inSeconds ... }
  FinalizedTrip finalize(DateTime endedAt) { ... durationSeconds: endedAt.difference(startedAt).inSeconds ... }
}
```

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-05-01 | Denial of Service (self) | sign-in cancel | mitigate | `GoogleSignInException(canceled)` caught → silent no-op, stay guest, dismiss sheet; never crashes (UI-SPEC error contract, Pitfall 1/5). |
| T-09-05-02 | Information Disclosure | error copy / logging | mitigate | Failure shows generic kCopySignInFailedBody; never logs token/credential (Security Domain V7). |

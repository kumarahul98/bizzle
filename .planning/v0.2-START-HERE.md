## Central technical finding
`flutter_background_service` cannot do continuous background GPS on iOS (15–30s every 15 min). The plan uses **geolocator `AppleSettings(allowBackgroundLocationUpdates: true, pauseLocationUpdatesAutomatically: false, activityType: automotiveNavigation)` + `UIBackgroundModes: location`** instead — a ~10-line platform branch in `tracking_service.dart`. Everything else is config. Zero package version changes needed.
